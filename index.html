Сумма массивов

Напишите функцию, которая получает в качестве аргументов 2 массива и возвращает массив, каждый элемент которого, равен сумме соответствующих элементов исходных массивов.

Если соответствующего элемента не существует (например, один из массивов короче) считать его равным 0.

Запрещается изменять исходные массивы.

Примеры:

args return
